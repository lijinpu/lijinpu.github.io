    • 总步骤

      • 第一步：观察数据：ggpairs()
      • 第二步：多元线性模型，使用Anova筛选出有解释力的自变量

    • 如果三大假设出了问题怎么办？

      • 第一步：解决异方差性问题

        • 常用方法：变换因变量，比如对因变量求对数

      • 第二步：解决线性问题

        • 常用方法：一般对自变量和因变量进行对数变换
        • 思考：为什么取对数？【真实世界往往是正态+头部效益，而头部效益是指数级】

  • ==常见的广义线性模型形式==

    1. 加入平方项：$y = k_1x+k_2x^2+b$ 解释：$x$每增加1，y增加$k_1+2k_2x$, 增速在加快/放缓
    2. 加入交乘项：$y = k_1x_1+k_2x_2+k_3x_1x_2+b$
      • 若$k_1$,$k_2$为正：
        • 若$k_3$显著为正，$x_1$,$x_2$有互补作用；
        • 若$k_3$显著为负，$x_1$,$x_2$有替代作用；
      • 思考：其他6种情况呢？【分析方法：求偏导】
    3. 对数变换：对谁取了对数变换，谁的单位就解释成百分比的变化
      • $In(y)=kx+b$, $x$增加1，$y$增加$100k%$
      • $y = k*In(x)+b$, $x$增加$1%$, $y$增加$0.01k$
      • $In(y) = k*In(x)+b$, $x$增加$1%$, $y$增加$k%$, “弹性”的概念

  • ==相关性热力图==——变量数量非常多时，看相关性的更==效率==的方法

    • 需要用到的包：dplyr,ggplot2,tidyverse,reshape2

  • ==预测误差==

    • $\frac{1}{n}\sum_{i=1}^n(y_i-y_i^`)^2$ #真实数据的值$-$使用模型预测的值

    • 预测误差越小越好

    • 两种测量预测误差的方法：直接法&间接法

      • 直接法：再随机收集一批新数据，看看模型在这些新数据中的预测情况

        • 缺点：收集新数据的成本太高
        • 解决
          • 将已有的数据分成2部分，比如70个建模，30个预测检验
          • 交叉验证（cross validation）

      • 间接法：==找一个类似于$R^2$的统计量，用它来估算预测误差==

        • 常用的间接统计量

          • AIC：Mallow’s $C_p$: $C_p=\frac{1}{n}(SSR+2d\cdot\hat{\sigma}^2)$

            • 注：
              • $n$是数据个数
              • $d$自变量个数
              • $\hat{\sigma}$是模型的均方误(mean squared error)，即summary()结果的最后一个部分
            • ==$C_p$的两个性质：==
            1. 随数据量增大，$C_p$会无限趋近于真实的预测误差
            2. $C_p$越小，模型误差越小，即预测精度越高
              • 注意：我们不需要收集任何新数据
    - **Bayesian Information Criterion(BIC): 贝叶斯信息标准**
      - 贝叶斯方法：不存在概率学派声称的“客观概率”。所谓的概率是==主观臆断和后验数据的信息之和
      - $BIC = \frac{1}{n}(SSR+k\cdot(ln(n))\cdot\hat{\sigma}^2)$
        - 注：
          - k是计算参数 = d+1
        - BIC越小，模型预测能力越好

  • ==模型选择==

    • 安装程序包leaps

      注：以上方法可以获得数个==局部最优==模型

    • 寻找全局最优模型

      注：$C_p$与$BIC$分别做标准时，选出的模型是不同的

      问：什么时候用哪个标准呢？

      答：==Elbow Rule==, 观察那个地方是下降最明显的。就使用下降最明显附近的点。我们要找的是比上一个模型有很大改进，比下一个模型又没有差很多的模型

  • 总结
    1. 目的：根据目前的的数据，找出最好的预测未来的模型
    2. 潜在自变量图示分析：自变量少的时候用ggpairs()自变量多的时候用==相关性热力图==
    3. 间接的预测能力标准：$C_p$和$BIC$
    4. 如何找所有的局部最优
    5. 从局部最优中如何找出全局最优
2.6 LASSO (Least Absolute Shrinkage and Selection Operator)【机器学习与模型筛选】

  • 预测模型时遇到的问题

    • 自变量数量太多，而数据个数又过少（少行多列）
    • 有没有办法进行数据降维，筛选出那些有意义的自变量
    • ==若想要成功跑回归，数据个数要大于自变量个数（维度的概念）==

  • 原理：

    • $min\sum_{i=1}^n(y_i-\beta_1X_{1i}-\beta_2X_{2i}-…-\beta_pX_pi)^2+\lambda\cdot(|\beta_1|+|\beta_2|+…+|\beta_p|)$
    • 平衡参数大小与残差平方和之间的关系
    • 找到最好的$\lambda$, 得到全局最优的模型
    • LASSO会把“不那么有用的”自变量的系数压缩成0
    • 特点
      • $\lambda$很小的时候，接近线性回归模型
      • $\lambda$很大的时候，所有系数都是0

  • 安装程序包glmnet并给定一个$\lambda$值跑lasso

  • 批量生产具有多个不同值$\lambda$的lasso模型

  • 看自变量系数如何随$\lambda$变化而变化

  • 找出99个模型中最好的一个

    • 最好=预测精度最高=预测误差最小

    • 如何计算？直接法！

    • 最大程度利用数据的方法：Cross Validation

      • 思路：比如共100数据，给定一个$\lambda$，90个来训练模型，剩下10个计算预测误差（要分别挑1-10，11-20,…,91-100这10组数据来计算预测误差，剩下90个数据训练模型，最后把这10组预测误差再做平均；即90个数据训练模型，而有100个数据来检验)

      • ==保证结果可复制，需要控制随机数种子==set.seed(1)

      • ==LASSO交叉检验==

      - 第一条竖线：MSE最小，全局最优 `lasso1.cv$lambda.min`
      - 第二条竖线：一倍标准误：`lasso.cv$lambda.1se`  “奥卡姆剃刀原则”：世界本身是简单的，可以让渡一部分精度，去选一个更简单的模型
  • LASSO的问题
    • 没有显著性
    • 结果是有偏的
  • 解决方法
    • relaxed LASSO
      • 思路：使用LASSO找出一组系数不为0的自变量后，再对这些自变量使用多元线性回归
      • ==即先用LASSO降维，而非直接作为预测模型==
2.7 脊回归与Elastic Net

  • Elastic Net

    $\frac{\sum_{i=1}^n(y_i-\beta_0-\beta_1X_{1i}-\beta_2X_{2i}-…-\beta_pX_{pi})^2}{2n}+\lambda\cdot{\frac{1-\alpha}{2}\sdot(\beta_1^2+\beta_2^2+…+\beta_p^2)+\alpha\cdot(|\beta_1|+|\beta_2|+…+|\beta_p|)}， 0\le\alpha\le1$

    • LASSO: L1 Norm

    • 脊回归：L2 Norm $\sum_{i=1}^n(y_i-\beta_0-\beta_1X_{1i}-\beta_2X_{2i}-…-\beta_pX_{pi})^2+\lambda\cdot(\beta_1^2+\beta_2^2+…+\beta_p^2)$

      • 脊回归不具有稀疏性，系数会等比例缩小，但不会缩小到0

  • 提醒：==LASSO和脊回归对单位非常敏感==，普通线性回归对单位不敏感

3.1 logit回归&分类

  • 因变量只有几个种类（最常见：0-1）

  • Logistic模型：二分因变量训练

  • logistic模型的本质：log odds $log\frac{P(A)}{1-P(A)}$

  • 为什么？我们希望把因变量从二元结构变成整个实数轴，将二元结构转变为从0-1的概率

  • $log\frac{P(A)}{1-P(A)}=kx+b$ $P(A)=\frac{e^{kx+b}}{1+e^{kx+b}}$

  • 模型的估算方法：最大似然估计

  • 计算机实现代码

    • ==模型标准：AIC越小越好==
    • 为什么结果中是z value？因为最大似然估计的一个前提是==数据是正态分布的==

  • 画图：箱线图

  • logit跑多元回归

  • 模型选择问题

  • 对概率设置分界线，反推x的界限（具体方法是人为找一个$P(A)=t$，而后反求x值）并画在上述散点图中

    • 犯两类错误

      • 假阳性（$1-Specificity$)：实际是0，被我们归为1

      • 真阳性($Sensitivity$)：实际是1，同时也被我们归为1

        • 真、假阳性会同增同减

        • 二者的关系图叫： ROC Curve

3.2 主成分分析(Principal Component Analysis)与聚类

  • 使用主成分分析==给数据降维==（当有多个相似变量时，希望提取出“最大公约数”），用一个指标概括多个指标

  • R内置画“两两散点图”的方法

  • 随机选50人的数据作为分析样本，并画图

  • 如何用一个变量代表两个变量？【**==做投影：提取第一主成分==**】

    • 找到一条线，距离这些点最近。每个点的投影（将直线看作数轴，向直线作垂线）即是新的变量
    • 要找到“数轴”的0点（均值）和单位长度（标准差）——将二维数据变成一维数据
      • 注：
        • N维数据最多可以提取出N个主成分
        • 第一主成分（最核心的趋势，例如：总成绩）与第二主成分（例如：成绩差值）是正交的（垂直），两个部分完全不相关

3.8 社会网络分析
  • 颠覆性创新 vs 边际性创新
  • 常用的社会网络指标：
    • 度数（degree）
      • 有几个邻居
      • degree(g1)
    • 聚类系数 Cluster Coefficient:
      • 有多少邻居也是你的邻居
      • 产生的“三角形”个数
      • transitivity(g1)
    • 结构洞 structural hole
      • 你的“经纪”能力：能力越强，数字越低
      • constraint(g1)
      • 公式
        • $p_{ij}=z_{ij}/(\sum_{q}z_{iq})$
        • $c_{ij}=(p_{ij}+\sum_{q}p_{iq}p_{qj})^2, i\ne q\ne j$
  • Burt与社会网络分析
    • 社会网络的问题：忽视个体差异，仅考虑所处的网络位置
    • 社会网络的意义：网络结构往往能够解决很多问题，比如“为什么女性在职场上晋升不如男性”
